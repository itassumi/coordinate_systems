# Лабораторна робота: Алгоритмічні та евристичні методи обчислення площі геометричних фігур

## Опис роботи
Порівняльний аналіз точного (метод Гауса/Shoelace) та наближеного (метод Монте-Карло) методів обчислення площі полігонів. Дослідження залежності точності та швидкодії від складності полігону та кількості ітерацій.

## Виконані завдання

### 1. Генерація та візуалізація полігонів
- Реалізовано генератор випадкових полігонів без самоперетинів
- Згенеровано полігони з різною кількістю вершин: 10, 50, 100
- Візуалізовано полігони та збережено зображення

### 2. Реалізація алгоритмів
- **Метод Гауса (Shoelace formula)** - точний аналітичний метод
- **Метод Монте-Карло** - наближений імовірнісний метод
- **Shapely** - еталонний метод (використовується для перевірки точності)

### 3. Дослідження точності методу Монте-Карло
- Проаналізовано залежність точності від кількості точок
- Побудовано графік збіжності методу

## Висновки

### 1. Метод Гауса (Shoelace)
- **Абсолютно точний** (0% похибки)
- **Швидкий** навіть для 1000 вершин
- **Ідеальний для простих полігонів**
- Час виконання лінійно залежить від кількості вершин

### 2. Метод Монте-Карло
- **Універсальний**, але **повільний**
- **Потребує 10,000+ точок** для точності <1%
- Швидкодія практично не залежить від складності полігону
- **Рекомендується для складних фігур**, де аналітичні методи важко застосувати

### 3. Shapely (бібліотечний метод)
- **Найшвидший** та **найнадійніший**
- **Рекомендований для практичного використання**
- Оптимізовані реалізації геометричних алгоритмів

## Практичні рекомендації

1. **Для простих полігонів** - використовуйте метод Гауса
2. **Для складних фігур** - використовуйте Shapely
3. **Метод Монте-Карло** - тільки якщо інші методи не підходять
4. **Для точності <1%** у методі Монте-Карло потрібно ~10,000 точок
5. **Для точності <0.1%** потрібно ~100,000 точок

## Висновок

Під час виконання лабораторної роботи було реалізовано та порівняно різні методи обчислення площі полігонів. Основні висновки:

1. **Метод Гауса** є оптимальним вибором для простих полігонів через високу точність та швидкодію
2. **Метод Монте-Карло** є універсальним інструментом, але вимагає значних обчислювальних ресурсів для досягнення високої точності
3. **Бібліотека Shapely** демонструє найкращі результати у практичному застосуванні

Результати експериментів підтверджують теоретичні положення про швидкість збіжності методу Монте-Карло та ефективність аналітичних методів для регулярних геометричних фігур.
